## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone this repository to your local machine
- git clone https://github.com/Soft-Magick/easy-biz-app.git
- cd easy-biz-app
- First, run the development server:

```bash
npm run dev
# or
yarn dev
```
